# Course Management Application

## Overview
This is a simple **Course Management Application** built using the **MERN stack with SQLite**.  
The application allows users to **register, log in, and manage courses** through a clean and secure system.

The project focuses on:
- Proper backend API design
- Authentication using JWT
- Secure data handling
- Clean frontend and backend separation

---

## Tech Stack

### Backend
- Node.js
- Express.js
- SQLite
- JSON Web Token (JWT)
- bcryptjs
- express-validator

### Frontend
- React.js
- Vite
- Axios
- React Router DOM

---

## Features

### User Authentication
- User registration with validation
- User login with email and password
- Password hashing using bcrypt
- JWT-based authentication for protected routes

### Course Management
- Create a new course
- View all available courses
- Update course details
- Delete a course
- Only authenticated users can perform course operations
